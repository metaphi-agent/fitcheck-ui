# fitcheck

Voice-based outfit recommendation app - "your best fit, every time"

## Features

- Clean, minimal mobile-first interface
- Voice input for natural interaction
- Suggest outfits based on context (dinner date, work, brunch, etc.)
- Inspired by modern iOS design patterns

## Design

This UI is based on a Figma design with:
- Black background for modern, sleek appearance
- White text and accent elements
- Voice-first interaction pattern
- Example prompts to guide user input

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Mobile-responsive design (optimized for iPhone 16 Pro dimensions)
